Sparse Autoencoders Reveal Interpretable Features in the Tahoe Single-Cell Foundation Model

Using mechanistic interpretability techniques to understand how the Tahoe-x1 model processes single-cell RNA sequencing data and encodes biological knowledge.

Introduction

Single-cell foundation models like Tahoe-x1 have demonstrated remarkable capabilities in understanding cellular biology, but their internal mechanisms remain largely opaque. At Lucent Cell, we applied sparse autoencoders (SAEs) to decompose the Tahoe model's learned representations into interpretable features that correspond to meaningful biological concepts.

Methodology

We trained TopK sparse autoencoders on intermediate layer activations from the Tahoe-x1 model using the SAELens framework. Our approach involved several key steps:

Model Setup

  • Foundation Model: Tahoe-x1 70M parameter model from HuggingFace
  • Hook Points: Extracted activations from MLP outputs at layers 0, 3, 6, 9, and 11
  • SAE Architecture: TopK SAE with k=64 active features and d_sae=16,384 total features
  • Training Data: PBMC 3k dataset with 2,700 cells and 32,738 genes

Training Process

# SAE Training Configuration
config = TopKTrainingSAEConfig(
    d_in=512,           # Tahoe hidden dimension
    d_sae=16384,        # SAE feature dimension
    k=64,               # Top-k sparsity
    learning_rate=3e-4,
    batch_size=4096
)

# Extract activations and train SAE
activations = extract_tahoe_activations(model, dataloader, layer=6)
sae = train_sparse_autoencoder(activations, config, steps=10000)

Key Findings

1. Cell Type-Specific Features

Several SAE features showed strong correspondence to known cell types. Feature 1247 activated specifically for T cells, with top-associated genes including CD3D, CD3G, and IL7R. Feature 892 was selective for monocytes, associating with CD14, FCGR3A, and S100A8.

2. Biological Pathway Encoding

Feature 2156 captured interferon response pathways, with strong associations to ISG15, IFIT1, and MX1. This suggests the model learns to represent coordinated gene expression programs that are fundamental to cellular function.

3. Disease-Associated Features

Remarkably, Feature 7834 showed enrichment for genes associated with inflammatory diseases through our validation against the Open Targets database, including TNF, IL1B, and CXCL8. This demonstrates the model's ability to encode disease-relevant biological patterns.

Implications for Research

Understanding Model Behavior

Our SAE analysis reveals that Tahoe-x1 learns biologically meaningful representations without explicit supervision. The model naturally organizes its internal features around cell types, pathways, and disease states that are fundamental to cellular biology.

Drug Discovery Applications

The interpretable features we extracted could guide drug target identification. Features that activate in disease-associated cellular states may reveal novel therapeutic targets or biomarkers for precision medicine approaches.

Model Improvement

Understanding which biological concepts the model captures (and which it misses) can inform training strategies for next-generation foundation models. Features with poor biological validation highlight areas where model architecture or training data could be improved.

Technical Implementation

Our complete pipeline is available as an open-source Python package that integrates with the Tahoe ecosystem. Key components include:

  • TahoeAdapter: Extracts activations from any Tahoe model layer
  • SAETrainer: Configurable sparse autoencoder training with SAELens v6
  • FeatureAnalyzer: Computes gene and cell-type associations
  • ValidationPipeline: Automated biological database validation
  • DashboardGenerator: Creates interactive HTML visualizations

Future Directions

This work opens several promising research directions:

  • Feature Steering: Manipulating SAE features to control model predictions
  • Cross-Model Analysis: Comparing features across different foundation models
  • Perturbation Studies: Using interpretable features to understand cellular responses
  • Clinical Validation: Testing feature-based insights in clinical datasets

Conclusion

Our sparse autoencoder analysis demonstrates that single-cell foundation models like Tahoe-x1 learn rich, interpretable representations of cellular biology. By making these representations transparent, we can better understand model behavior, validate biological insights, and guide the development of more effective AI systems for biological research.

This work represents a crucial step toward trustworthy AI in biology, where model predictions are not just accurate but also interpretable and actionable. As foundation models become increasingly powerful, mechanistic interpretability will be essential for translating their capabilities into real-world therapeutic advances.
